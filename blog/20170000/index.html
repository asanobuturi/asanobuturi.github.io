編集方法

  1. GitHub Desktopをインストール。
    https://desktop.github.com/
  2. GitHub Desktopを開いて，File→Clone repository
  3. URLのタブに，「https://github.com/asanobuturi/asanobuturi.github.io」を入力。
  4. Cloneを押す。
    クローンされると，マイドキュメント の GitHub フォルダ内にasanobuturi.github.ioができているはず。
    それを編集し，GitHub Desktopでコミット，プッシュすれば，数十秒後くらいに更新される。
